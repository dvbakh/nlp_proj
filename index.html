Поиск по корпусу

Описание проекта

Цель проекта - создание корпуса текстов Братьев Стругацких с функцией поиска.
Формат поиска таков:

  • лемма/словоформа (ищет вхождения во всех формах);
  • словоформа/фраза в двойных кавычках (ищет только заданную форму);
  • лемма и POS-тег (ищет все формы, отмеченные данным тегом).
Список и обозначения тегов можно посмотреть тут.

Примеры запросов:
  • поиск - нужно найти все предложения, где слово встречается в любой форме
  • “поиска” - нужно найти предложения только с этой формой
  • знать+NOUN - нужно найти все предложения, где встречается существительное “знать”
  • NOUN - найти все предложения с существительными
